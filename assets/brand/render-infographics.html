unmachined · how it works

Two lines of defense.
Neither is optional.

Models converge on the same center. unmachined does not ask the model to “sound more human.” It checks known tells first, then refuses to replace one monoculture with another.

01
LINE 01

Severity-tiered catalogs

Deterministic scanners score prose and UI source before any model self-judgment.

  • critical 20 · major 10 · minor 3
  • score 0–100, fail at 40
  • exit code 1 for CI and hooks
02
LINE 02

Variety requirement

A fix becomes a new tell the moment every output converges on the same alternative.

  • two runs should not share a voice
  • layout skeletons must not clone
  • banning “delve” just moves the cluster

Priority when effort splits: text first · design & layout equal second · branding last

unmachined · command surface

Six verbs. One skill.

The skill is the command. The first argument selects the workflow. Hosts without a leading slash use the same words in natural language.

unmachined · pipeline blueprint

Two pipelines. Shared discipline.

Surface detection picks the path. Scan first, then model judgment. Hard gates stay offline and reproducible.

Text pipeline MAIN PRIORITY
  1. 01
    Load catalogs text-tells.md · voice-and-copy.md for microcopy
  2. 02
    Draft or edit Write the claim first. Vocabulary comes second.
  3. 03
    Run scan_text.py Score >= 40 blocks fix and build delivery
  4. 04
    Self-critique once What still reads generated? One revise pass.
  5. 05
    Honesty gate Real numbers only. No invented quotes or metrics.
Design pipeline UI · LAYOUT
  1. 01
    Load design rules design-tells · design-system · stack-rules when needed
  2. 02
    Run scan_ui.py Grep-able tells before visual judgment
  3. 03
    Lock structure + tokens Layout skeleton first. OKLCH tokens. One bold risk.
  4. 04
    Pre-emit critique Six axes, 1-5. Any axis under 3 triggers a rewrite.
  5. 05
    Hard gates WCAG 2.2 AA · reduced motion · focus · 320-768px

unmachined · install blueprint

One skill directory.
Four host symlinks.

Clone once. Point each agent host at the same tree. The skill supplies the command; no separate command files.

# global install
$ git clone https://github.com/0xNyk/unmachined.git
$ cd unmachined
$ ln -s "$(pwd)" ~/.claude/skills/unmachined
$ ln -s "$(pwd)" ~/.codex/skills/unmachined
$ ln -s "$(pwd)" ~/.cursor/skills/unmachined
$ ln -s "$(pwd)" ~/.hermes/skills/unmachined

# scanners: Python 3.8+, zero deps
